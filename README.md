fun minecraft mod with some random features. download the jar once and it updates itself automatically.

breakdown:
1. sloppy joe villager: sells sloppy joes. has a special workbench
2. perspective lens: allows you to shrink/grow blocks and mobs. only cosmetically grows blocks past a 5x scale and below a 1x scale.
3. horse planes: lets you fly a horse.

## Crafting Recipes

**Shapeless**

| Result | Ingredients | Output |
|---|---|---|
| Sloppy Joe | Bread + Cooked Beef | 2x |

**Shaped** — `/` = next row, space = empty slot

| Result | Grid | Key |
|---|---|---|
| Perspective Lens | `IGI / G G / IGI` | I = Iron Ingot, G = Glass Pane |
| Glass Jar | `GGG / G G / GGG` | G = Glass |
| Pilot Saddle | `LIL / ISI / LLL` | L = Leather, I = Iron Ingot, S = Saddle |
| Aviator Hat | `LLL / LFL` | L = Leather, F = Feather |
| Aviation Goggles | `GLG` (single row) | G = Glass Pane, L = Leather |
| Pilot Helmet | `III / ILI` | I = Iron Ingot, L = Leather |
| Horse Plane Ticket | ` I  / PFP /  I ` | I = Iron Ingot, P = Paper, F = Feather |
| Joe Counter | `PPP / PBP / PPP` | P = Oak Planks, B = Bread |
